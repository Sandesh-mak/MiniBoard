# MiniBoard – Task Management System

MiniBoard is a **Java Swing-based task management application** that allows teams to manage and track tasks in a **Kanban-style board**.  
It supports creating, updating, deleting tasks, tracking deadlines, exporting tasks to CSV, and logging activity.

---

## Features

- **Login System**: Admin and Employee roles  
- **Kanban Board**: 3 columns – To Do, In Progress, Done  
- **Task Management**: Add, edit, delete tasks  
- **Task Priority**: LOW, MEDIUM, HIGH  
- **Due Date Tracking**: Overdue task alerts  
- **Search Functionality**: Filter tasks by title, description, or assignee  
- **Export Tasks**: Save tasks to CSV  
- **Activity Log**: Track all task changes  
- **Employee Assignment**: Assign tasks to employees  
- **Role-based Permissions**: Admins have full access; Employees can update task status only
